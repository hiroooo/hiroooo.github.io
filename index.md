# hiroooo apps

個人開発アプリ群のポータル。各アプリの LP は sub-path で配信。

- [Sublog](/sublog/) — サブスク管理アプリ

## 運用メモ

- このリポは GitHub Pages root (`https://hiroooo.github.io/`)
- `app-ads.txt` は AdMob Verify 用、全アプリ共通の publisher ID で 1 ファイル運用
- 新アプリは `hiroooo/<slug>` リポを作って `<slug>.github.io` ではなく **このドメイン下の sub-path** で運用すること（`hiroooo.github.io/<slug>/`）
